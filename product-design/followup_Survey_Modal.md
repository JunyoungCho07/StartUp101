## 추가 follow-up 명령 (선택 사항)

### 명령 A — 인라인 마이크로 설문 (메인 페이지 Pain Mirror 섹션 아래)
1문항짜리 매우 가벼운 설문을 추가하고 싶다면:
```
Pain Mirror 섹션 아래에 인라인 단답 설문 1개 추가:
- 헤드라인: "여러분은 어떠셨어요?"
- 객관식 1문항: "최근 1개월 셀프 계산 중 직원 호출한 적 있나요?" 
  - 예 / 아니오 / 셀프 계산을 안 함
- 응답 즉시 결과 카운터 노출: "방문자 N명 중 X%가 '예'라고 답했습니다"
- 응답 후 "베타 사전등록하기" CTA 노출 (Hero CTA로 스크롤)
- 응답은 localStorage에 저장 (중복 응답 방지)
```

이 인라인 설문은 **social proof + 1차 funnel 가속** 용도. 단, 메인 폼 전환율 측정과 분리해서 A/B로 운영해라.

### 명령 B — Thank You 페이지 분리 (선택)
모달 대신 별도 thank-you 페이지로 가고 싶다면:
```
폼 제출 후 모달 대신 /thank-you 라우트로 redirect.
/thank-you 페이지에 동일한 6문항 설문을 페이지 형태로 배치.
URL: /thank-you?email={email}&store={store_pref}
```

## 측정 KPI (모달 도입 후 추가 지표)

| 단계 | 지표 | Threshold |
|---|---|---|
| 등록 → 모달 노출 | 100% (자동) | — |
| 모달 노출 → Q1 답변 | Q1 응답률 | 60% 이상 |
| Q1 → Q6 (완주율) | Survey completion rate | 35% 이상 |
| Q6 인터뷰 참여 동의율 | **Commitment metric** | **15% 이상** |
| Q6에서 휴대폰 + 시간 모두 입력 | Strong Commitment | 8% 이상 |

> **15% Commitment**가 의미하는 것: 발표 슬라이드 10에서 "Commitment 미충족"이라고 인정한 약점을 메울 수 있는 지표.

## Mom Test 위반 자기 점검 체크리스트

- [ ] Q1: 과거 행동(최근 1개월) — ✓ 통과
- [ ] Q2: 과거 행동(가장 최근 호출 이유) — ✓ 통과
- [ ] Q3: 과거 데이터(시간 측정) — ✓ 통과
- [ ] Q4: 과거 감정 회상 — ✓ 통과
- [ ] Q5: 과거 지출 데이터 — ✓ 통과 (가장 강력한 신호)
- [ ] Q6: 약속(미래) — ⚠ 의향이 아닌 시간 차단으로 받음 = 통과

> Q5는 특히 중요. "있나요?"가 아닌 "지출한 적 있나요?"로 물어 **실제 행동 데이터**를 캡처. 이 데이터가 "이 사람은 비대면·속도에 돈을 쓸 사람인가"의 1차 지표.

## 데이터 분석 방법 (수집 후 1주차)

### 1차 분석: Pain Hierarchy 검증
- Q2 복수선택 결과 카운트
- 셀프 선호자 21명 중 17명(81%) 불편 구조와 일치하는지 비교
- 만약 "바코드 < 성인인증 < 쿠폰" 순서가 뒤집히면 **솔루션 우선순위 재배치 필요**

### 2차 분석: Willingness to Pay (간접)
- Q5에서 "둘 이상 지출" 응답자 vs "없음" 응답자의 Q6 인터뷰 참여율 비교
- 유의미한 차이 있으면 → 페이팅(paying) segment 식별 가능

### 3차 분석: Pain → Commitment 전환
- Q3 "5분 이상" 답변자의 Q6 참여율 vs "1분 이내" 답변자
- 시간 비용이 클수록 인터뷰 동의율이 높아야 가설 정합

### 분석 결과로 다음 iteration 결정
- Commitment 15% 미만 → 솔루션 가설 재검토 필요 (제품 가설 약함)
- Q4 감정 응답에 "짜증" / "포기" 등 강한 부정 감정 비율 < 30% → Pain 강도 부족

## 직접적 경고

이 모달이 너희에게 주는 진짜 자산은 **인터뷰 풀(pool)**이다. Q6에 휴대폰 + 시간 슬롯 적은 사람들 = 다음 라운드 심층 인터뷰 대상. 이게 지금 너희 팀이 **n=4 인터뷰의 한계를 깰 수 있는 유일한 합법적 지름길**이다.

설문 데이터 자체보다, 설문이 만들어내는 **인터뷰 deal flow**가 100배 가치 있다는 걸 잊지 마라.
